Creating the Bootable USB

Now we're going to install Clover. You can download it here.

Installing Clover
  1. When you get to the Installation Type tab, wait.
  2. Click on Change Install Location.
  3. Select the USB dirve and continue.
  4. Now click on customize.
  5. Check Install for UEFI Booting Only, and under Drivers64UEFI, check OsxSptioFix2Drv-64 and PartitionDXE-64.

Now you can install Clover. But before you boot, you still have to configure Clover.
